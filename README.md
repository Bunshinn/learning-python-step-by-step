接触python也有将近10年的时间了, 但一直只能说是处于入门的水平吧.   
因为工作的关系, 偶尔会用python解决一下遇到的问题, 也都是现学现卖. 以解决问题为目标, 能够很快的上手, 但对于整体掌握一门语言来说是不够的. 所以, 希望在这里能更系统的讲解一下python编程的入门知识. 既是对以往的学习成果作一个总结, 也希望作为分享培训的资料, 能够帮助到小伙伴们共同成长吧.
